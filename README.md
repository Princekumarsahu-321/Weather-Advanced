# 🌦️ Advanced Weather App

A responsive weather application built using **HTML, CSS, and JavaScript** with real-time Weather API integration. The application provides live weather details such as temperature, humidity, wind speed, latitude, and longitude for any searched location.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature updates
* 💨 Wind speed information
* 💧 Humidity details
* 📍 Latitude and longitude tracking
* 📱 Responsive design for mobile and desktop
* ⚡ Fast API data fetching with JavaScript

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (ES6)
* Weather API

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/weather-app.git
```

2. Open the project folder

```bash
cd weather-app
```

3. Run the project

* Open `index.html` in your browser

---

## 🔑 API Setup

1. Get a free API key from a weather service provider like OpenWeatherMap.
2. Add your API key inside `script.js`

```javascript
const apiKey = "YOUR_API_KEY";
```
